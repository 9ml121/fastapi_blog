## 🎯 使用指南

### 开发时
1. 首先查看 **项目规范** 了解开发标准
2. 参考 **项目管理** 了解当前进展
3. 查阅 **参考文档** 获取具体设计信息

### 学习时
1. 按阶段阅读 **学习资源** 中的材料
2. 结合 **参考文档** 理解设计决策
3. 查看 **项目规范** 了解最佳实践

### 查找问题
- **开发规范问题** → `standards/`
- **技术原理问题** → `learning/`
- **设计细节问题** → `reference/`
- **进展状态问题** → `project/`

| 分类 | 用途 | 特点 | 更新频率 |
