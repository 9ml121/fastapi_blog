# 前端开发学习计划

## 项目概述

**目标**：为现有 FastAPI 博客后端开发完整的前端应用

**产品定位**：**个人博客 + 文章社交**二合一平台

**核心功能**：

1. 实现自研 Markdown 编辑器，用于个人博客文章管理
2. 实现文章社交系统，用于学习现代前端登录、注册、网页布局和前后端通信等技能

**设计理念**：

- 「Markdown 编辑器」 参考 `Typora`、`Obsidian` 等标杆产品的专业设计
- 「文章社交系统」 参考 `Medium` 和 `知乎` 等标杆产品的设计
- 采用前端微服务设计，「Markdown 编辑器」和「文章社交系统」做成可组合的独立功能

**技术栈**：Vue 3 + Vite + TypeScript + Pinia + Vue Router + Axios + marked + Lucide Icons

**核心原则**：

- ✅ 实战驱动：Agent 辅助用户进行模块功能设计、实现步骤分解、代码讲解等，用户完成最终编码
- ✅ 渐进式学习：从基础到进阶，逐步深入
- ✅ 教学互动：理论讲解 + 实操任务 + 答疑复盘

## 学习路径（4个阶段）

### Phase 1：Vue 3 基础 + 环境搭建（3-5天）

**学习目标**：

- 掌握 Vue 3 核心概念（响应式、组件、生命周期）
- 搭建前端开发环境（Vite + TypeScript）
- 完成基础页面布局和路由配置

**核心任务**：

1. 开发环境搭建
   - 初始化 Vite + Vue 3 + TypeScript 项目
   - 配置 ESLint + Prettier

2. Vue 3 核心概念学习
   - 组件基础（模板语法、响应式数据）
   - 组合式 API（ref、reactive、computed）
   - 生命周期钩子

3. 基础页面开发
   - 创建基础布局组件（Header、Footer、Sidebar）
   - 配置 Vue Router
   - 实现基础页面（首页、示例列表页）

**验收标准**：

- ✅ 开发服务器正常运行
- ✅ 基础页面路由跳转正常
- ✅ 组件响应式数据更新正常

**前置知识**：

- HTML/CSS/JavaScript 基础
- npm/pnpm 包管理器基本使用

**学习资源**：

## 学习建议

### 1. 学习节奏

- **Phase 1**（3-5天）：快速入门，掌握基础
- **Phase 2**（2-3周）：深入实战，攻克难点
- **Phase 3**（2-3周）：完整功能，融会贯通
- **Phase 4**（可选）：进阶优化，生产就绪

### 2. 理论与实践结合

- 每天先学理论（30-60分钟），再实战编码（2-3小时）
- 遇到问题先查文档，再问 AI
- 重要概念及时记录到 `docs/learning/` 目录

### 3. 代码质量保障

- 使用 ESLint + Prettier 保持代码风格
- 组件拆分合理，单一职责原则
- 写注释，特别是复杂逻辑
- 定期代码审查

### 4. 进度追踪

- 每完成子任务更新 `process.md`
- 遇到困难创建 `task.md` 详细记录实现步骤
- Phase 完成后更新 `overview.md`

### 5. 不要追求完美

- Phase 1-3 先实现功能，不纠结样式
- Phase 4 再优化体验和性能
- 渐进式改进，避免过早优化
