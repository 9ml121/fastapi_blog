# 顶部导航栏需求文档

> **功能模块**：AppNavbar
> **版本**：v1.0（简化版）

---

## 1. 功能概述

顶部导航栏是网站的核心导航组件，提供品牌展示、页面导航和用户操作入口。

---

## 2. 核心需求

### 2.1 功能需求

| 需求 | 说明 | 优先级 | 版本 |
|-----|------|-------|------|
| 品牌展示 | Logo + 网站名称，点击返回首页 | ⭐⭐⭐ | v1.0 |
| 导航菜单 | 博客、干中学、好东西 | ⭐⭐⭐ | v1.0 |
| 当前位置高亮 | 当前页面对应的菜单项高亮 | ⭐⭐⭐ | v1.0 |
| 用户状态显示 | 未登录显示"登录"，已登录显示头像 | ⭐⭐⭐ | v1.0 |
| 主题切换 | 深色/浅色模式切换 | ⭐⭐ | v1.0 |
| 搜索入口 | 搜索图标占位（暂不实现功能） | ⭐ | v1.0 |
| 头像下拉菜单 | 个人中心、设置、退出登录 | ⭐⭐ | v1.1 |
| 响应式适配 | 移动端汉堡菜单 | ⭐⭐ | v1.1 |
| 菜单项下拉 | Categories 展开子菜单 | ⭐ | v2.0 |

### 2.2 菜单项定义

| 菜单项  | 路由         | 说明        |
| ---- | ---------- | --------- |
| Logo | `/`        | 网站图标 + 名称 |
| 博客   | `/blog`    | 文章列表      |
| 干中学  | `/learn`   | 教程/课程     |
| 好东西  | `/goodies` | 工具/资源     |

### 2.3 用户状态场景

| 状态 | 显示 | 交互 |
|-----|------|------|
| 未登录 | "登录" 按钮 | 点击跳转 `/login` |
| 已登录 | 用户头像 | 点击展开下拉菜单（v1.1） |

---

## 3. 交互设计

| 交互 | 效果 |
|-----|------|
| Logo 点击 | 跳转首页 `/` |
| 菜单项点击 | 跳转对应页面 |
| 菜单项 Hover | 文字颜色变化 |
| 当前页面 | 菜单项高亮（颜色 + 下划线） |
| 主题切换 | 切换深色/浅色模式 |

---

## 4. 样式规格

| 属性 | 值 |
|-----|---|
| 高度 | 64px |
| 背景 | 半透明 + 毛玻璃效果 |
| 定位 | `position: sticky; top: 0;` |
| 层级 | `z-index: 100;` |
| 内容最大宽度 | 1200px |

---

## 5. 版本规划

| 版本 | 功能 |
|-----|------|
| **v1.0（当前）** | 基础导航 + 用户登录按钮 + 主题切换 |
| v1.1 | 头像下拉菜单 + 响应式适配 |
| v2.0 | 菜单项下拉 + 搜索功能 |
