### 1.2 布局技术方案

| 区域 | 布局方式 | CSS 关键代码 |
|-----|---------|-------------|
| 整体页面 | 垂直 Flex | `display: flex; flex-direction: column; min-height: 100vh;` |
| 内容区容器 | 水平居中 | `max-width: 1200px; margin: 0 auto; padding: 0 24px;` |
| 主内容 + 侧边栏 | CSS Grid | `display: grid; grid-template-columns: 1fr 340px; gap: 48px;` |

### 1.3 响应式断点

| 断点 | 宽度 | 布局变化 |
|-----|------|---------|
| Desktop | ≥ 1024px | 两栏布局（主内容 + 侧边栏） |
| Tablet | 768-1023px | 侧边栏隐藏，单栏布局 |
| Mobile | < 768px | 单栏布局，导航栏折叠为汉堡菜单 |

## 2. 顶部导航栏 (AppNavbar)

### 2.1 结构设计

```
┌─────────────────────────────────────────────────────────────────┐
│  [Logo]   博客   干中学   好东西    [🔍 搜索...]    [🌙] [👤]   │
└─────────────────────────────────────────────────────────────────┘
    ↑         ↑                          ↑          ↑    ↑
  品牌区   导航菜单                     搜索框    工具区  用户区
```

### 2.2 菜单项定义

| 菜单项 | 路由 | 图标 | 说明 |
|-------|------|------|------|
| Logo | `/` | 自定义 | 网站图标 + 名称 |
| 博客 | `/blog` | 📝 | 文章列表 |
| 干中学 | `/learn` | 🎓 | 教程/课程 |
| 好东西 | `/goodies` | 🎁 | 工具/资源 |
| 个人中心 | `/profile` | 👤 | 用户头像/下拉菜单 |

### 2.3 功能区设计

| 功能 | 位置 | 实现方式 |
|-----|------|---------|
| 搜索框 | 中间 | 输入框 + 快捷键提示 (⌘K) |
| 主题切换 | 右侧 | 图标按钮（🌙/☀️） |
| 用户头像 | 最右 | 未登录显示"登录"按钮，已登录显示头像 + 下拉菜单 |

### 2.4 样式规格

| 属性 | 值 |
|-----|---|
| 高度 | 64px |
| 背景 | `rgba(255, 255, 255, 0.8)` + `backdrop-filter: blur(8px)` |
| 阴影 | `0 1px 0 rgba(0, 0, 0, 0.1)` |
| 定位 | `position: sticky; top: 0;` |

## 3. 右侧边栏 (AppSidebar)

### 3.1 结构设计

```
┌────────────────────────────────┐
│  📌 热门文章                    │
│  ├── 1. 文章标题...            │
│  ├── 2. 文章标题...            │
│  └── 3. 文章标题...            │
├────────────────────────────────┤
│  🏷️ 标签云                      │
│  [Vue] [React] [CSS] [动画]    │
│  [TypeScript] [性能优化]        │
├────────────────────────────────┤
│  👤 关于作者                    │
│  [头像]                        │
│  昵称：xxxx                    │
│  简介：前端开发者...            │
└────────────────────────────────┘
```

### 3.2 组件拆分

| 组件 | 说明 | 数据来源 |
|-----|------|---------|
| `PopularPosts.vue` | 热门文章列表（Top 5） | API: `/posts?sort=views&limit=5` |
| `TagCloud.vue` | 标签云 | API: `/tags` |
| `AuthorCard.vue` | 作者简介卡片 | 静态配置 或 登录用户信息 |

### 3.3 样式规格

| 属性 | 值 |
|-----|---|
| 宽度 | 340px（固定） |
| 间距 | 各模块间距 24px |
| 卡片样式 | 圆角 12px，背景 `var(--color-bg-card)` |

## 4. 文章卡片 (PostCard)

### 4.1 结构设计

```
┌──────────────────────────────────────────────────────────────┐
│  [Vue]              ← 分类标签（可选）                        │
│                                                              │
│  CSS 动画从入门到精通                                         │
│  ↑ 标题（大字、加粗）                                         │
│                                                              │
│  本文将带你深入理解 CSS 动画的核心概念，包括 transition、     │
│  animation、@keyframes 等，并通过实战案例掌握...             │
│  ↑ 摘要（2-3 行，超出省略）                                   │
│                                                              │
│  2024年12月23日  ·  阅读 5 分钟                               │
│  ↑ 元信息（日期、阅读时长）                                   │
└──────────────────────────────────────────────────────────────┘
```

### 4.2 Props 定义

```typescript
interface PostCardProps {
  id: string              // 文章 ID
  title: string           // 标题
  summary: string         // 摘要（前 150 字）
  category?: string       // 分类
  tags?: string[]         // 标签（可选）
  publishedAt: string     // 发布日期
  readingTime?: number    // 阅读时长（分钟）
  coverImage?: string     // 封面图（可选）
}
```

### 4.3 交互效果

| 状态 | 效果 |
|-----|------|
| 默认 | 无阴影，白色背景 |
| Hover | 轻微上浮 `translateY(-4px)` + 阴影 |
| 点击 | 跳转到文章详情页 `/blog/{id}` |

### 4.4 样式规格

| 属性 | 值 |
|-----|---|
| 内边距 | 24px |
| 圆角 | 12px |
| 背景 | `var(--color-bg-card)` |
| 边框 | `1px solid var(--color-border)` |
| 卡片间距 | 24px |

## 5. 组件文件结构

```
src/

│   ├── layout/                 # 布局组件
│   │   ├── AppLayout.vue       # 页面骨架

│   │   └── AppSidebar.vue      # 右侧边栏
│   │
│   ├── sidebar/                # 侧边栏子组件

│   │   ├── TagCloud.vue        # 标签云
│   │   └── AuthorCard.vue      # 作者卡片
│   │
│   └── post/                   # 文章相关组件
│       └── PostCard.vue        # 文章卡片
│
├── views/
│   └── HomeView.vue            # 首页（使用 AppLayout）
│
```

## 6. 实现难度评估

| 组件 | 难度 | 预估时间 | 关键技术点 |
|-----|------|---------|-----------|
| **AppLayout** | ⭐⭐ | 1 小时 | CSS Grid、插槽 |
| **AppNavbar** | ⭐⭐⭐ | 2-3 小时 | Flexbox、下拉菜单、响应式 |
| **AppSidebar** | ⭐⭐ | 1 小时 | 卡片布局 |
| **PostCard** | ⭐⭐ | 1 小时 | Hover 动画 |
| **响应式适配** | ⭐⭐⭐ | 2 小时 | 媒体查询、汉堡菜单 |

**总预估**：8-10 小时（约 1-2 天）

## 7. 实现顺序建议

| 顺序 | 任务 | 说明 |
|-----|------|------|
| 1 | AppLayout.vue | 先搭骨架，用占位符填充内容 |
| 2 | AppNavbar.vue | 实现基础导航，不含下拉菜单 |
| 3 | PostCard.vue | 静态卡片展示 |
| 4 | 首页组装 | HomeView.vue 集成上述组件 |
| 5 | AppSidebar.vue | 添加热门文章、标签云 |
| 6 | 响应式适配 | 移动端布局调整 |
